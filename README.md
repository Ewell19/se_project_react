# WTWR (What to Wear?)

## About the Project

WTWR is a responsive React application that helps users decide what to wear based on real-time weather conditions. The app fetches live weather data for Martinsburg, WV and displays clothing recommendations that are appropriate for the current temperature and time of day.

## Features

- **Live Weather Data**: Displays real-time temperature and location from the OpenWeatherMap API
- **Dynamic Weather Card**: Background changes based on weather conditions (hot/warm/cold) and time of day (day/night)
- **Smart Clothing Recommendations**: Filters and displays clothing items appropriate for current weather conditions
- **Interactive Item Cards**: Click on clothing items to view details in a modal
- **Add Clothes Modal**: Interface for adding new clothing items (functionality to be implemented in future sprints)
- **Responsive Design**: Works seamlessly across different screen sizes

## Technologies Used

- **React**: Component-based UI library
- **Vite**: Fast build tool and development server
- **JavaScript ES6+**: Modern JavaScript features
- **CSS3**: Custom styling with BEM methodology
- **OpenWeatherMap API**: Real-time weather data
- **Git & GitHub**: Version control and deployment
